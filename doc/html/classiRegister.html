Detailed Description

Defines a "register" in the Wi-11 machine.

The methods present in this interface are meant to mimic the functionality of the Wi-11 machine, allowing for simplified execution of the instructions therein. This interace class will serve as a base from which the general purpose registers and program counter of the Wi-11 can be defined.
