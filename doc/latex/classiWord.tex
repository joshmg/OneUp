\subsection{Detailed Description}
Defines a \char`\"{}word\char`\"{} of data on the Wi-\/11 Machine. The methods present in this inteface are meant to mimic the functionality of the Wi-\/11 machine, allowing for simplified execution of the instructions therein. As the size of a \char`\"{}word\char`\"{} depends on the architecture, classes implementing this interface should define the word length to be 16 bits in length.

\end{DoxyPrecond}
\begin{DoxyReturn}{Returns}
True $<$=$>$ 1, False $<$=$>$ 0.
\end{DoxyReturn}
The number of the bits starts at zero and rises into the more significant bits. \begin{DoxyParagraph}{Examples:}
If the object holds a value of 4 (0...100 in binary): num\mbox{[}2\mbox{]} = 1.\par
 If it holds a value of 1 (0...001 in binary): num\mbox{[}0\mbox{]} = 1.\par
 If it holds a negative value (Starting with a 1 in 2's complement): num\mbox{[}15\mbox{]} = 1. 
\end{DoxyParagraph}
